Research Experience

  • Study of Spanning tree with Maximum number of leaves

    Spanning Tree of a graph is a tree where all nodes are connected. A graph might have multiple spanning tree. A maximum leaves spanning tree (MLST) is a tree whose number of leaves is maximum. Finding MLST of a graph is a NP-hard. It is also APX-hard, which means it's approximation ratio is bounded by a constant. In this work, we tried to find an approximate algorithm for the MLST of a graph.

  • Chakma Handwritten Characters recognition and Multipurpose Dataset

    The Chakma language is a endangered language spoken by the Chakma people from Bangladesh. In this work we've provided a novel multipurpose dataset on Handwritten Chakma Characters, a baseline model and various benchmark and statastical data.

  • Multilingual machine translation for Chakma

    In this work we provided a small chakma to bangla translation corpus and some monolangual data. We experimented with statastical models, as well as NN models for translation purpose.
